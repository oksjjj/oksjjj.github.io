## **읽게 된 계기**

1990년대부터 코딩은 취미로 쭉 즐겨 왔으나,  
웹에 대해서는 전혀 관심도 없었고, 모르는 것에 대한 막연한 두려움도 있었다  

그러다 업무 효율화를 위한 프로그램을 실행 파일로 만들어서 필요한 사람들에게 배포한 적이 있었는데,  
변경 사항이 생길 때마다 실행 파일로 만들어서 배포해야 한다는 것이 너무나 불편했다  
(물론 프로그램 자동 업데이트를 개발하면 되겠지만, 여건상 그건 불가능했다)  

그래서 웹으로 개발하면 좋겠다는 생각을 하게 되었다  

파이썬 Flask를 공부하고, Chat GPT의 도움을 받아서 어찌 어찌 웹 페이지를 만들긴 했는데,  
나는 아무 것도 모른 채, 책에 있는 코드, Chat GPT가 알려준 코드를 복사해서 붙여 넣는 수준에 불과했다.  

웹에 대해서 제대로 배워 보고 싶다는 생각이 들었다

## **소감**

### **코드 실습이 매우 쾌적했다**

일단 visual studio code와 vscode의 extension인 Live Server만 설치하면 모든 실습이 가능하다  

그리고, 저자님이 모든 실습 예제를 압축 파일로 만들어서 자료실에 올려두셨기 때문에  
해당 파일만 받아서 실습하면 된다  

특히나, 각 챕터별로 실습해야 할 코드들만 빈칸으로 되어 있기 때문에  
챕터와 관계없는 다른 HTML 코드들을 일일이 타이핑할 필요가 없어서 빠른 실습이 가능하다  


### **반응형 웹이 무엇인지 알게 해주었다**

같은 웹사이트를, PC로 볼 때, 스마트폰으로 볼 때, 다른 모양으로 나오는 원리를 알게 되었다  
미디어 쿼리 하는 기능을 이용해서, 디스플레이 사이즈에 따라 각각 다른 CSS를 적용하는 것이다  
이 책을 보고 나니, 다른 책, 특히 bootstrap 관련 책을 볼 때 더 잘 이해를 할 수 있게 되었다  


### **얇지만 두루두루 다루고 있다**

기본 HTML, CSS, 자바스크립트에 관해 최대한 많은 내용을 다루려고 노력하신 흔적이 보인다  
이제는 좀 더 심화 과정의 책을 더 잘 공부할 수 있을 것 같다
